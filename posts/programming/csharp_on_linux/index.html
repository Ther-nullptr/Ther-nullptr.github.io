Csharp on Linux

2022-06-21
#Csharp 

dotnet build 生成一个.NET工程。目录下必须有.sln文件或.csproj文件。

dotnet publish 生成一个.NET工程的发布版本。目录下必须有.sln文件或.csproj文件。

dotnet run 启动一个.NET工程。目录下必须有.csproj文件。

dotnet clean 清除一个.NET工程的所有编译文件。目录下必须有.sln文件或.csproj文件。

dotnet list package 查看一个.NET工程的所有依赖包。目录下必须有.sln文件或.csproj文件。

dotnet list reference 列举项目的所有依赖项。目录下必须有.csproj文件。

dotnet add package <PACKAGE_NAME> --version <VERSION> 添加一个依赖包。目录下必须有.csproj文件。

publish命令和build命令最大的区别在于：publish命令将应用程序的依赖项，从NuGet缓存复制到输出文件夹中。
